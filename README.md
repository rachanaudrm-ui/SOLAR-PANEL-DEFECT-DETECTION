# Features
- Image upload for solar panel analysis
- Gemini AI integration for defect detection
- Responsive UI
- TypeScript + Vite for fast development

View AI Studio prototype: https://ai.studio/apps/35aded73-93b7-40a3-ab4e-b2341813ef65

# Run Locally

**Prerequisites:** Node.js 18+

1. Clone & install:
   ```
   npm install
   ```
2. Create `.env.local` with your Gemini API key:
   ```
   GEMINI_API_KEY=your_key_here
   ```
3. Start dev server:
   ```
   npm run dev
   ```

# Backend Setup
- Supabase project with `supabase-blueprint.sql`
- Tables for users, images, analysis results

# Tech Stack
```
Frontend: Vite + React + TypeScript + Tailwind
AI: Google Gemini API
Backend: Supabase (Postgres + Auth)
Deployment: Vercel/Netlify
```
